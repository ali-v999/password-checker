# 🔐 Password Strength Checker

This is my first cybersecurity project.

## 📌 Description
This program checks how strong a password is based on:
- Length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## ⚙️ How it works
The user enters a password and the program gives feedback:
- Weak ❌
- Medium ⚠️
- Strong ✅

## 🚀 How to run
1. Make sure Python is installed
2. Open terminal in the project folder
3. Run:
   python main.py

## 🛠️ Technologies used
- Python
- Regex (re module)

## 📚 What I learned
- Basics of Python
- Using functions
- Regular expressions
- Password security concepts

## ✨ Features
- Password strength checker
- Detects common weak passwords
- Password generator
- Multiple strength levels (Weak → Very Strong)
